#  Circles Chat

Circles Chat is a **real-time private messaging application** designed for secure and confidential communication within trusted circles. It also includes an **AI Assistant** to help users interact, respond, and enhance their chat experience intelligently.

---

## Features

-  Secure private messaging
-  Real-time chat using Socket.io
-  circle-based communication
-  AI Assistant integration for smart replies and assistance
-  Instant message delivery
-  Responsive UI for mobile and desktop

---

##  AI Assistant

Circles Chat includes an **AI-powered assistant** that helps users with:
- Smart chat responses
- Conversation assistance
- Enhancing communication experience

---

## Tech Stack

-  React.js
-  TypeScript
-  Tailwind CSS
-  Socket.io (real-time communication)
-  Axios (API requests)
